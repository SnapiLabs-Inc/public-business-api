## Requirements

* Node v10 and above



## Service Setup

First clone the repo and then `cd` into it.

`git clone https://github.com/SnapiLabs-Inc/public-business-api.git`

`cd public-business-api`

### Install Modules

`npm install`


### Configuration



### Start the service

`node bin/www`
